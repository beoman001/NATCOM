About the Creator

The Mind Behind
the Language

Beon Binesh is the creator of NATCOM — the world's first natural English programming language. Built entirely using human intuition and powered by Google Gemini AI and Antigravity AGY by Google DeepMind, NATCOM represents the future of human-computer interaction.


What began as a question — "Why do humans learn the machine's language?" — became a revolutionary compiler that understands English as code. NATCOM doesn't just interpret instructions. It thinks, self-heals, and compiles natural sentences into native binaries and JavaScript — making programming accessible to every human mind on Earth.

Paying Tribute

Standing on the
Shoulders of Giants

NATCOM pays tribute to COBOL — the first English-like programming language, created by Admiral Grace Murray Hopper in 1959. Like COBOL, NATCOM believes code should read like English. But where COBOL required rigid structure and was built for business punch cards, NATCOM is free-form, modern, and compiles to native binaries and JavaScript.

Evolution

The NATCOM Timeline

v1.0 — The Idea

"Why do humans learn the machine's language?" This question birthed the first iteration of NATCOM — a parser that understood simple English statements and converted them into raw logic.

v1.5 — The Compiler Matrix

Introduction of the Sovereign Compiler Toolchain. NATCOM evolved to parse complex variables, loops, math, and logic, seamlessly targeting highly optimized Native C binaries.

v2.0 — Object Oriented English

A massive breakthrough. Added support for defining Functions, Classes, and Objects entirely in natural English prose. Multi-target compilation (C and JS) introduced for Web and Desktop execution.

v2.1 — AI Genesis (Current)

NATCOM Studio now features a built-in AI Code Generator via the Google Gemini API. Users prompt in English, and the AI writes valid NATCOM code perfectly. 4-letter GTA-style cheat codes added.
